# College Football Predictor 2024

## Overview

This project aims to predict outcomes of college football games using machine learning. It focuses on two main prediction tasks:

1. Win probability: Predicting the probability of each team winning a specific matchup.
2. Score prediction: Predicting the final score for both teams in a specific game.

## Features

- Data scraping from the [CollegeFootballData.com API](https://collegefootballdata.com/)
- Data cleaning and processing

## Planned

- Exploratory data analysis
- Feature engineering
- Machine learning model development
- Model evaluation and comparison with betting lines
- (Planned) Interactive dashboard for predictions
